# Infinite Hidden Markov Model and Infinite Wishart Mixture Model
Gibbs Sampler with Split-Merge Moves for the Infinite Hidden Markov Model (IHMM) (on top of Juergen Van Gaels IHMM toolbox http://mloss.org/software/view/205/) and the Infinite Wishart Mixture Model (IWMM) implemented in MATLAB

Van Gael, J. (July, 2010). The Infinite Hidden Markov Model 0.5. Retrieved from http://mloss.org/software/view/205/
